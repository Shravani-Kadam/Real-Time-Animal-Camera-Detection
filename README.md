# Animal Detection Backend

Flask-based backend using YOLO for animal detection.

## Features
- YOLO-based animal detection
- REST APIs for frontend integration
- CSV logging of detections

## Tech Stack
- Python
- Flask
- YOLO (Ultralytics)
- OpenCV

## Deployment
Deployed on Render using GitHub integration.
